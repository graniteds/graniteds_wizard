
Granite Data Services Wizard (http://www.graniteds.org)
--------------------------------------------------------------

* Provide:

  A generic project wizard system, with specific templates for GraniteDS.

* Requirements:

  Eclipse 3.4+, JDK 6+.

* Installation:

  Close Eclipse, remove any older version of the plugin, drop the new
  org.granite.builder_x.x.x.jar into your <ECLIPSE_HOME>/plugins directory,
  then restart Eclipse.
